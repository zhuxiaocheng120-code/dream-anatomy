# 微信小程序备案与审核文案说明

## 建议提交说明

本小程序是一款个人梦境记录、睡眠感受记录与 AI 辅助文字整理工具。

用户可以记录梦境文字、睡眠感受和醒来后的个人想法。用户主动提交文字后，系统可对该文字进行摘要整理、情绪词识别、意象关键词整理，并生成开放式自我反思问题，便于用户保存和回顾个人记录。

本小程序不提供解梦、算命、占卜、吉凶判断、未来预测、通灵或其他封建迷信相关服务，不对梦境符号作固定含义解释，也不根据梦境判断用户的命运、健康状况或现实事件。

本产品不是医疗、心理诊断或心理治疗服务，AI 生成内容仅作为记录整理和自我反思参考。

## 小程序展示层调整

- 小程序用户可见名称使用：Dream Anatomy 梦境手札。
- 主要功能名称使用：AI 整理梦境。
- 结果页展示为：梦境线索卡、文字线索整理、意象关键词和开放式自我反思问题。
- 深度记录入口继续显示“正在开发中”，不触发 AI 请求。
- 小程序展示层会对 AI 返回文本做合规降风险替换，但不会修改本机保存的原始 `reportContent`。

## 保留边界

- 不删除 AI 整理功能。
- 不删除结果页或梦境线索卡。
- 不修改 Web 端文案、服务端提示词、数据库结构或 DeepSeek API 调用。
- 不新增微信登录、云同步、支付或会员功能。
